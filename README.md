# BinBuz

**BinBuz** — a lightweight, private tool to generate formatted **Business Identification Number (BIN)** details for Business Manager verification and record‑keeping.

> Simple. Local. Private. Built for social‑media verification workflows.

---

## Features
- Command-line step‑by‑step input for BIN details.
- Outputs a clean, human‑readable formatted block (terminal + optional text file save).
- Lightweight Python script, zero external API calls (works offline).
- Easy to customize and extend (CSV batch mode, GUI, verification backend — future).
